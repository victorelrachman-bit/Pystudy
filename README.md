#  PyStudy

PyStudy é uma plataforma simples de gestão de estudos, onde o usuário pode registrar seu tempo de estudo, quantidade de questões feitas e acertos por matéria, e visualizar tudo em gráficos interativos.

---

##  Funcionalidades

- Cadastro de usuário com senha criptografada (bcrypt)
- Login com sessão (Flask session)
- Registro de estudos por matéria:
  - Tempo de estudo
  - Questões feitas
  - Questões acertadas
  - Data do estudo
- Visualização de dados em gráficos (Chart.js)
- Separação por matérias:
  - Matemática
  - Inglês
  - Programação
- Proteção de rotas (somente usuário logado acessa dados)
- Logout seguro

---

### Frontend
